# 执业画像 · 何时做测评与自评

> 与 [[001-我的执业画像]]、[[002-执业画像自评问卷]] 配合使用。

---

## 建议触发复测的时机

| 时机 | 说明 |
|------|------|
| **首次建立画像** | 填一次完整自评问卷 + 任选做大五完整量表（见问卷内链接），建立基线。 |
| **每季度** | 快速过一遍问卷中的边界与 DUA 打分，有变化则更新 `我的执业画像.md` 并记一行「最近自评记录」。 |
| **连续高压期后** | 如连续加班、密集庭审、多人情案叠加后 1～2 周内，复查情绪边界与人情边界评分。 |
| **执业阶段变化** | 实习 → 独立执业、转所、主要业务领域切换时，整卷重填。 |
| **接案策略调整** | 当你主动决定「多接 / 少接 / 不接」某类案件时，同步更新风险偏好与回避类型，并做一次自评。 |

---

## 大五人格等外部量表

完整大五（IPIP 等）**不必**按季度重做；建议 **每年 1 次**，或与执业阶段变化同步。日常以本库 **自评问卷** 为主即可。

---

## 在画像中的记录方式

在 [[001-我的执业画像]] 文末「最近自评记录」表中填写：**日期、触发条件、变化摘要**；必要时注明「当日完成大五 IPIP」等。
